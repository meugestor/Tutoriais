# CASAOS
Manual de instalação !

## RECOMENDAÇÃO DE CONFIGURAÇÃO
> UBUNTU 20.04
> 6GB RAM

> PREPARE SUA VPS

```bash
sudo apt update && apt upgrade && apt install git && sudo apt install curl
```

```bash
apt --fix-broken install 
```

> INSTALE O CASAOS
```bash
curl -fsSL https://get.casaos.io | sudo bash
```
